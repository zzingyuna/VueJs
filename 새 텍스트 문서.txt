v-text : 텍스트만 적용
v-html : html태그도 적용 가능, 스크립트 보안 취약
v-bind : 요소 객체의 속성들을 바인딩하기 위해 사용
v-model : 양방향 데이터 바인딩이 필요한 경우 사용
v-if : 조건에 부합되지 않으면 렌더링을 하지 않는다 (v-if, v-else-if, v-else가 있다)
v-show : 조건에 부합되지 않으면 display 속성을 none으로 바꾼다
v-for : 반복 렌더링
:key="" : 기존 DOM 요소의 데이터를 변경하거나 할때 v-for 바인딩 되는 데이터에 키값을 부여
v-pre : 컴파일 하지 않고 안에 든 문자열 그대로를 출력
v-one : 단 한번만 렌더링 되도록 한다
v-cloak : 화면 초기에 컴파일 되지 않은 템플릿은 나타나지 않도록 할 수 있다.
배열의 filter 메서드는 배열의 아이템 중 조건을 만족하는 아이템을 모아서 새로운 배열을 만들어 리턴하는 기능
